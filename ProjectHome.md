**Welcome to OpenBiomind-GUI: GUI for OpenBiomind!**

# About OpenBiomind-GUI #
OpenBiomind-GUI provides  a graphical interface to execute the tasks provided by [OpenBiomind](http://code.google.com/p/openbiomind/). It provides wizards for all the tasks provided by OpenBiomind. In addition, it also provides wizards that chain multiple tasks together so that they can be executed together. For more details please [look at the feature list](FeatureList.md).

The first version of OpenBiomind-GUI was developed as a [Google Summer of Code 2008](http://code.google.com/soc/2008/) project by [Bhavesh Sanghvi](http://www.cs.iastate.edu/~bsanghvi) with [Murilo Saraiva de Queiroz](http://code.google.com/u/muriloq/) (official) and [LÃºcio de Souza Coelho](http://code.google.com/u/lucioc/) as mentors. The application has been developed using [Java SE 6](http://java.sun.com/javase/6/) and [Eclipse](http://www.eclipse.org/) 3.4 RCP. If you are a developer, then you may wish to [learn about development environment](DevelopmentEnvironment.md).
